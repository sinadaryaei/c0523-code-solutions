# css-fonts-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- Name two types of units that can be used to adjust `font-size` in CSS.
  Absolute units are used to set the precise size.
  While relative units set the elements size relative to the what it is displayed on.

- What CSS property controls the font used for the text inside an element?
  The `font-family` is used for the text inside an element.
